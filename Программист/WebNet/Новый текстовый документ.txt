Simplex-ная связь - Когда вещает кто-то один, остальные слушают.
Half-Duplex - Когда каждый может и вещать и слушать, но в конкурентной среде и по очереди.
Full-Duplex - Когда каждый может и вещать и слушать. При этом есть отдельные среды для вещания и прослушивания.
Unicast- Когда кто-то вещает целенаправленно кому-то конкретному.
Broadcast - Когда один сразу вещает всем.
Multicast - Когда кто-то вещает только одной какой-то группе. Остальные вещания не получают.
LAN - Local Area Network - локальная сеть, находящаяся под администрированием одного человека или команды.
WAN - Wide Area Network - Внешняя сеть относительно к рассматриваемой LAN сети.
Коммутация пакетов - Вариант одновременного установления связи между сколь угодным количеством участников с помощью передачи небольших кусков данных.
Коммутация каналов - Вариант установления связи между двух участников с помощью резервирования канала связи между ними.

Роутер - устройство, которое точечно пересылает пакет в определенный порт, основываясь на L3 заголовке.
Switch (коммутатор) - устройство, которое точечно пересылает пакет в определенный порт, основываясь на L2 заголовке.
Таблица коммутации (MAC address table) в коммутаторе - Специальная таблица, в которой сопоставляются номера портов коммутатора с МАС адресами хостов, которые располагаются за этими портами.
unknown destination unicast - Это когда коммутатор рассылает пакет во все порты, если Destination MAC в этом пакете неизвестен коммутатору (нет с ним записи в МАС address table).
Spannign Tree Protocol - Специальный протокол для предотвращения петель (и broadcast шторма) в сетях.
IP адрес хоста - Уникальное 32-ух битное значение, принадлежащее хосту в сети. Обычно записывается как четыре числа(от 0 до 255), разделенных точкой.
Хаб - устройство, которое повторяет входящий сигнал во все остальные порты.
broadcast - Это когда коммутатор рассылает пакет во все порты, если Destination MAC равен FF-FF-FF-FF-FF-FF.
Шлюз по умолчанию на хосте (Default Gateway) - Это IP адрес роутера, на который хост должен слать пакеты в чужие IP сети.
IP сеть - Специальный диапазон IP адресов, заданный с помощью маски из единиц и нулей.
Таблица ARP (Address Resolution Protocol) - Специальная таблица, в которой сопоставляются IP адреса с МАС адресами.
MTU - Maximum Transmit Unit - максимальная длина payload'a после заголовка L2 в пакете.
